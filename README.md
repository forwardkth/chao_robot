# chao_robot
The third generation Robot with advanced sensors and perception, localization and navigation algorithms
## ECU:
JetsonTX2, Beaglebone White or Black
## Sensors
IntelRealsense D435i, Incremental encoder, Ultrasonic
## Software platform
ROS kinetic
